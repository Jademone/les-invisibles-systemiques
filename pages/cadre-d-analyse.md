# Cadre d’analyse

Cette page ne décrit pas une méthode reproductible.
Elle pose un **cadre de lecture**.

## Ce que ce cadre permet
Comprendre ce que produisent réellement les décisions numériques,
au-delà des intentions, des discours et des indicateurs.

## Trois angles d’observation

### Gouvernance
- Où se prennent les décisions
- Ce qui est centralisé, délégué ou invisibilisé
- Ce qui est prescrit versus ce qui est piloté

### Équilibres
- Ce qui est déplacé plutôt que résolu
- Les arbitrages implicites (temps, charge, attention)
- Les zones de saturation ou de fragilisation

### Mécaniques humaines
- Les adaptations informelles
- Les contournements nécessaires pour « que ça tienne »
- Les coûts humains rarement reconnus

## Ce que cette page n’est pas
- Un mode d’emploi
- Une grille prête à l’emploi
- Une recette applicable sans contexte

Les analyses publiées sur le site s’appuient sur ce cadre,
mais leur construction détaillée n’est pas exposée ici.
